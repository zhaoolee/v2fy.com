## 我写了个Node.Js爬虫，自动化爬取壁纸娘壁纸(已开源)

运行环境: nodejs 10.0.0 及更高版本

```javascript
git clone https://github.com/zhaoolee/bilibili-wallpaper.git
cd bilibili-wallpaper
npm i
npm start
```
